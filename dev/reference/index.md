## Datasets

- [`smartStd`](https://nutriverse.io/anthrocheckr/dev/reference/smartStd.md)
  : Standardisation test data - sample 1
- [`smartStdLong`](https://nutriverse.io/anthrocheckr/dev/reference/smartStdLong.md)

  subjects.

  : Dataset from a standardisation exercise done in Liberia in
  preparation for a coverage survey.
