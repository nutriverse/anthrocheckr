# Standardisation test data - sample 3

Example dataset from the Standardized Monitoring and Assessment of
Relief and Transitions (SMART) capacity building toolbox found at
\<http://smartmethodology.org/survey-planning-tools/smart-capacity-building-toolbox/\>.
This dataset is the same as the `smartStd` dataset but has been kept in
the original wide format.

## Usage

``` r
smartWide
```

## Format

A data frame with 67 columns and 10 rows

## Source

Standardized Monitoring and Assessment of Relief and Transitions (SMART)
capacity building toolbox found at
\<http://smartmethodology.org/survey-planning-tools/smart-capacity-building-toolbox/\>.
