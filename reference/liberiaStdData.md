# Dataset from a standardisation exercise done in Liberia in preparation for a coverage survey.

Dataset from a standardisation exercise done in Liberia in preparation
for a coverage survey.

## Usage

``` r
liberiaStdData
```

## Format

A data frame with 7 columns and 744 rows:

\| \*\*Variable\*\* \| \*\*Description\*\* \| \| :— \| :— \| \| \*eid\*
\| Enumerator ID \| \| \*cid\* \| Child ID \| \| \*round\* \|
Standardisation test round \| \| \*weight\* \| Weight measurement \| \|
\*height\* \| Height measurement \| \| \*muac\* \| Mid-upper arm
circumference measurement \| \| \*oedema\* \| Bilateral pitting oedema
\|
